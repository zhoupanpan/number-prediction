插件名称: NamePrediction

需求概述: 开发数字预测与投入产出比计算功能，具体包括以下内容：

1. 数字组合生成与输入
   - 支持生成/输入由0-9组成的5位不重复数字组合
   - 提供随机生成和手动输入两种数字组合获取方式

2. 预测结果判定规则
   - 以福彩3D每日开奖结果作为判定依据
   - 当输入数字组合与开奖结果一致时判定为"预测正确"，否则为"预测错误"

3. 投入产出计算模型
   - 基础投入：每组数字组合投注金额为10元
   - 产出规则：
     - 预测正确时产出160元
     - 预测错误时产出0元
     - 支持倍数投注（投入金额=10元×倍数，产出金额=160元×倍数）
   - 累计计算：需实时统计历史累计投入金额与累计产出金额

4. 数据记录功能
   - 自动记录所有数字组合（包括生成方式、具体数值）
   - 完整保存每次预测的投入金额、倍数、产出金额及累计盈亏数据



## 技术栈

- **前端框架**: Vue 3 + Vuex + Vue Router
- **构建工具**: Vite
- **样式**: CSS3 + 渐变背景 + 毛玻璃效果
- **数据存储**: LocalStorage
